## 当前平衡快照

- 普通模式当前是调试基线，不是最终商业化难度。
- 关键 Boss 当前生命：第一章 `forge_bishop` 100，第二章 `storm_archon` 116，最终 Boss `nexus_heart` 128。
- 篝火休息恢复最大生命的 40%。
- 状态规则：`vulnerable`、`weak`、`frail` 是消耗型层数；`burn` 回合开始伤害并递减；`strength` 长期保留。
- 当前正式矩阵覆盖 `3 角色 x 4 挑战 x 256 = 3072` 场配对完整跑团：
  - 普通、挑战 1、挑战 2、挑战 3 的三角色平均胜率分别为 `6.5% / 3.8% / 2.0% / 1.2%`。
