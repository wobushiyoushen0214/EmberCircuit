# 遗物位图生产设计

## 统一视觉语言

- 单件遗物居中，三分之四视角，轮廓优先于微小纹理。
- 材质以黑锻铁、旧黄铜、炭石、玻璃和少量皮革/纸张为主。
- 通用能量使用克制的余烬橙；回声石使用青白冷光；灰烬念珠使用白焰作为角色识别。
- 不添加底座、地面、卡框、徽章、文字或外溢粒子，避免在 HUD 中形成杂边。
- alpha 可见包围盒目标为画布面积的 `0.30-0.82`，四周保留透明安全边。

## 资源处理

1. 生图模型输出 `1024x1024`、纯 `#ff00ff` 色键背景的 PNG。
2. 使用统一色键工具执行 soft matte 与 despill；若有残边，只允许一次 `edge-contract 1` 修正。
3. 使用 Lanczos 缩放为 `512x512 RGBA`，不锐化、不加阴影、不改色。
4. 检查透明四角、alpha 包围盒、主体占比和 32px 缩略图可读性。
5. 通过资源清单接入，不在 UI 代码中硬编码具体遗物路径。

## 页面验证

- 战斗 HUD：六件遗物容量、38px 图标和工具提示保持可用。
- 战斗奖励/宝箱/商店：42px 主图不遮挡标题、副标题和描述。
- 图鉴：58px 艺术栏完整显示透明轮廓，锁定条目仍不泄露图像。
