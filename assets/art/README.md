# Art Asset Manifest

This folder contains placeholder production slots for `EmberCircuit`.

## Current Placeholder Assets

- `player_ember_exile.svg`: first playable character placeholder.
- `enemy_soot_raider.svg`: normal enemy placeholder.
- `enemy_ash_hound.svg`: normal enemy placeholder.
- `enemy_forge_bishop.svg`: chapter one boss placeholder.
- `card_attack_frame.svg`: attack card frame direction.
- `card_skill_frame.svg`: skill card frame direction.
- `card_power_frame.svg`: power card frame direction.
- `relic_placeholder.svg`: relic icon placeholder.
- `potion_placeholder.svg`: potion icon placeholder.
- `potion_placeholder.svg.import`: Godot import metadata for the potion icon placeholder.

## Replacement Rules

- Keep filenames stable until data keys are migrated.
- Final card art should preserve readable name, cost, type, and description areas.
- Enemy art should export with transparent backgrounds.
- Boss art should be readable at 320 px width in combat.
