# Audio Asset Manifest

The project now ships deterministic first-pass WAV sound effects and lightweight
music loops generated by `tools/generate_audio_assets.py`. `AudioManager.gd`
loads these files first and falls back to generated tones only if an SFX asset is
missing.

## Rules

- Keep event IDs stable; code and tests refer to these names.
- Regenerate first-pass assets with `python3 tools/generate_audio_assets.py`.
- Commercial release assets should replace the WAV files in-place with licensed
  or custom-designed SFX/BGM while preserving filenames.
- UI sounds should remain short and quiet; combat, boss phase, victory and defeat
  can carry more weight but should not mask readable battle feedback.
- Music loops are intentionally short first-pass placeholders. They exist to
  prove context switching, settings and asset replacement slots before final
  licensed or custom composition work.
